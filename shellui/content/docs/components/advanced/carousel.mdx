---
title: Carousel
description: Image and content carousel with navigation controls
---

The Carousel component provides a slideshow interface for cycling through content. It supports autoplay, looping, navigation arrows, and dot indicators.

## Installation

```bash
shellui add carousel
```

## Basic Usage

```razor
<Carousel>
  <CarouselContent>
    <CarouselItem>
      <div class="p-6 text-center">Slide 1</div>
    </CarouselItem>
    <CarouselItem>
      <div class="p-6 text-center">Slide 2</div>
    </CarouselItem>
    <CarouselItem>
      <div class="p-6 text-center">Slide 3</div>
    </CarouselItem>
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## With Images

```razor
<Carousel>
  <CarouselContent>
    @foreach (var image in images)
    {
      <CarouselItem>
        <img src="@image.Src" alt="@image.Alt" class="w-full rounded-lg object-cover" />
      </CarouselItem>
    }
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
  <CarouselDots />
</Carousel>

@code {
    private List<ImageItem> images = new()
    {
        new("/images/slide1.jpg", "Mountain landscape"),
        new("/images/slide2.jpg", "Ocean sunset"),
        new("/images/slide3.jpg", "Forest trail")
    };

    record ImageItem(string Src, string Alt);
}
```

## Card Carousel

Display cards in a scrollable carousel:

```razor
<Carousel>
  <CarouselContent>
    @foreach (var product in products)
    {
      <CarouselItem Class="md:basis-1/2 lg:basis-1/3">
        <Card>
          <CardHeader>
            <CardTitle>@product.Name</CardTitle>
            <CardDescription>@product.Price</CardDescription>
          </CardHeader>
          <CardContent>
            <p class="text-sm">@product.Description</p>
          </CardContent>
        </Card>
      </CarouselItem>
    }
  </CarouselContent>
  <CarouselPrevious />
  <CarouselNext />
</Carousel>
```

## Autoplay

Enable automatic slide transitions:

```razor
<Carousel Autoplay="true" AutoplayInterval="5000" Loop="true">
  <CarouselContent>
    <CarouselItem><div class="p-6">Slide 1</div></CarouselItem>
    <CarouselItem><div class="p-6">Slide 2</div></CarouselItem>
    <CarouselItem><div class="p-6">Slide 3</div></CarouselItem>
  </CarouselContent>
  <CarouselDots />
</Carousel>
```

## API Reference

### Carousel

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `Autoplay` | `bool` | `false` | Enable automatic transitions |
| `AutoplayInterval` | `int` | `3000` | Milliseconds between slides |
| `Loop` | `bool` | `false` | Loop back to start after last slide |
| `ChildContent` | `RenderFragment` | `null` | Carousel sub-components |

### Sub-Components

| Component | Description |
|-----------|-------------|
| `CarouselContent` | Container for carousel items |
| `CarouselItem` | Individual slide wrapper |
| `CarouselPrevious` | Previous slide button |
| `CarouselNext` | Next slide button |
| `CarouselDots` | Dot indicators for each slide |

## Accessibility

The Carousel component includes:
- `role="region"` with `aria-roledescription="carousel"`
- `aria-label` on each slide
- Previous/Next buttons with `aria-label`
- Keyboard navigation with arrow keys
- Autoplay pauses on hover and focus
- Respects `prefers-reduced-motion`

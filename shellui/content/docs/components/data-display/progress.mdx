## Overview

The Progress component displays the completion state of an operation as a horizontal bar. It accepts a `Value` (0â€“100), a customizable `Height`, and merges additional classes via `Shell.Cn()`.

## Installation

```bash
shellui add progress
```

## Usage

### Basic Progress

```razor
<Progress Value="50" />
```

### With Label

```razor
<div class="space-y-2">
  <div class="flex justify-between text-sm">
    <span>Upload Progress</span>
    <span>@progress%</span>
  </div>
  <Progress Value="@progress" />
</div>

@code {
    private int progress = 45;
}
```

### Custom Height

```razor
<Progress Value="75" Height="1rem" />
```

### Animated Loading

```razor
@if (isLoading)
{
  <div class="space-y-2">
    <div class="flex justify-between text-sm">
      <span>Processing...</span>
      <span>@progress%</span>
    </div>
    <Progress Value="@progress" />
  </div>
}

@code {
    private bool isLoading = true;
    private int progress = 0;

    protected override async Task OnInitializedAsync()
    {
        for (int i = 0; i <= 100; i += 10)
        {
            progress = i;
            StateHasChanged();
            await Task.Delay(500);
        }
        isLoading = false;
    }
}
```

### Multiple Progress Bars

```razor
<div class="space-y-4">
  <div class="space-y-2">
    <div class="flex justify-between text-sm">
      <span>Storage</span>
      <span>@storage%</span>
    </div>
    <Progress Value="@storage" />
  </div>
  <div class="space-y-2">
    <div class="flex justify-between text-sm">
      <span>Bandwidth</span>
      <span>@bandwidth%</span>
    </div>
    <Progress Value="@bandwidth" />
  </div>
</div>

@code {
    private int storage = 65;
    private int bandwidth = 30;
}
```

### Custom Styling

```razor
<Progress Value="80" Class="bg-muted" />
```

## API Reference

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `Value` | `int` | `0` | Progress value from 0 to 100 |
| `Height` | `string` | `"0.5rem"` | CSS height of the progress bar |
| `Class` | `string?` | `null` | Additional CSS classes merged via `Shell.Cn()` |

## Accessibility

- Rendered with `role="progressbar"`, `aria-valuenow`, `aria-valuemin`, and `aria-valuemax`.
- Screen readers announce the current progress percentage.
- Visual fill width reflects the `Value` for sighted users.
